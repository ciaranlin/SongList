## 公共主页

公众访客看到的主页，展示你的歌单和信息卡片。

### 功能说明

#### 1. 顶部横幅区域
- **头像**：圆形大头像（可在配置中自定义）
- **标题**：你的频道名称或标语
- **副标题**：简短描述
- **信息卡片**：鼠标悬停或点击展开

#### 2. 歌单表格
- **搜索功能**：快速查找歌曲
- **语言筛选**：国语、日语、英语、其他
- **点歌功能**：点击任意歌曲行可复制"点歌"指令
  - 格式：`点歌 歌名`（可自定义）

#### 3. 响应式设计
- **桌面版**：侧边栏式布局
- **手机版**：完整竖向布局，自动调整

## 配置管理

访问 `/config` 进入配置面板。有5个主要标签页：

### 标签 1️⃣：主题

#### 背景颜色
- 点击色块选择颜色
- 或输入十六进制颜色代码（如 `#FF69B4`）

#### 文字颜色
- **自动模式**：根据背景自动调整为黑或白
- **手动模式**：自定义文字颜色

#### 入口图标
- **显示配置入口**：显示齿轮图标进入配置
- **显示歌单入口**：显示歌单入口

#### 管理密码
- 修改后需要重新登录

---

### 标签 2️⃣：横幅

#### 头像设置
- **输入URL**：直接输入图片链接
- **上传图片**：点击"上传头像"按钮选择本地文件

头像会显示为圆形，建议使用正方形图片。

#### 文案设置
- **标题**：主要标题（如频道名）
- **副标题**：副标题描述
- **提示文字**：悬停提示（如"移入展开"）

#### 头图卡片展示
- **展示模式**：
  - "始终显示"：卡片始终可见
  - "滚动触发"：滚动到该区域时显示
  - "悬停触发"：鼠标悬停时显示
  
- **卡片间距**：卡片之间的距离（px）
- **动画时长**：展开/收起动画时间（ms）

#### 悬停热区设置（仅在"悬停触发"模式）
- **启用精准热区**：只在头像/标题区域触发
- **触发区域**：选择敏感区域
- **显示悬停提示**：鼠标悬停时显示提示文字
- **退出延迟**：鼠标离开后的延迟（防止闪烁）

---

### 标签 3️⃣：卡片 ⭐ 新功能

这是新增的卡片编辑功能！

#### 添加卡片
点击右上角"+ 添加卡片"按钮创建新卡片。

#### 编辑卡片内容

##### 标题
卡片的大标题，显示在最上方。

##### 内容
卡片的描述文本，支持多行。

##### 🆕 卡片图片
**全新功能！** 为卡片添加漂亮的背景图片。

- **上传图片**：点击"上传图片"按钮选择本地文件
- **输入URL**：直接输入网络图片链接
- **预览**：图片预览会实时显示

###### 推荐图片规格
- 宽高比：16:9 或 4:3
- 文件大小：<1MB
- 格式：JPG、PNG、WebP

#### 链接管理

##### 添加链接
点击"+ 添加"按钮，为卡片添加可点击的链接按钮。

每个链接包括：
1. **名称**：链接显示的文字（如"推特"、"油管"）
2. **URL**：链接地址（如 https://twitter.com/your_handle）

##### 🆕 图标选择 ⭐
**全新功能！** 从多个图标库中选择链接的图标。

**可用图标：**

| 分类 | 图标 |
|------|------|
| **社交媒体** | 推特、油管、B站、Facebook、Instagram、LinkedIn |
| **通用** | 全球、GitHub、邮件、电话、位置、链接、分享 |

**如何使用：**
1. 点击链接下方的"图标选择"下拉菜单
2. 从列表中选择想要的图标
3. 保存配置

###### 图标预览
在配置页面右侧的实时预览中，可以看到图标效果。

## 歌单管理

访问 `/yu` 进入歌单管理页面。

### 功能

#### 搜索和筛选
- 按歌曲名、歌手名搜索
- 按语言筛选（国语/日语/英语/其他）
- 按"舰长可点"筛选

#### 添加歌曲
点击"+ 新增"按钮，填写：
- **歌曲名**：必填
- **歌手名**：必填
- **语言**：必填（国语/日语/英语/其他）
- **备注**：可选（如"热门歌曲"）
- **舰长可点**：勾选允许舰长点这首歌

#### 编辑歌曲
点击列表中的歌曲，弹出编辑对话框修改信息。

#### 删除歌曲
点击歌曲行的删除按钮（垃圾桶图标）。

## 新功能详解

### 💎 功能 1：卡片图片上传

#### 为什么需要卡片图片？
- 让卡片更吸引人
- 更好地展现内容
- 提升视觉效果

#### 如何使用？

1. **进入配置** → **卡片** 标签
2. **选择卡片**，找到"卡片图片"区域
3. **三种方式添加图片：**
   
   **方式 A：上传本地文件**
   ```
   1. 点击"上传图片"按钮
   2. 选择本地图片文件
   3. 自动上传并预览
   ```
   
   **方式 B：输入URL**
   ```
   1. 在图片URL输入框中粘贴网络图片链接
   2. 按回车或点击其他地方
   3. 图片自动加载预览
   ```
   
   **方式 C：删除图片**
   ```
   清空输入框，卡片图片会被移除
   ```

4. **实时预览**：右侧预览区实时显示效果
5. **保存**：点击页面右上角"保存"按钮

#### 图片要求

| 项目 | 要求 |
|------|------|
| 文件大小 | ≤ 5MB |
| 格式 | JPG、PNG、WebP |
| 分辨率 | 建议 1920x1080 或更高 |
| 宽高比 | 16:9 推荐 |

#### 示例

```
Good:  📸 320x180px, 清晰的风景照
Bad:   📸 100x100px, 模糊的小图
Bad:   📸 10MB, 超大文件
```

## 常见问题

### Q：图片上传失败？
A：
1. 检查文件大小（≤5MB）
2. 检查文件格式（JPG/PNG/WebP）
3. 检查网络连接
4. 尝试更换图片文件

### Q：更改了图片为什么看不到？
A：
1. 点击页面右上角"保存"按钮
2. 如果还看不到，清除浏览器缓存（Ctrl+F5）

### Q：能用动图（GIF）作为卡片图片吗？
A：不建议。虽然技术上可以，但会影响性能。建议使用静态图片。

### Q：图片太大怎么办？
A：
1. 使用图片压缩工具（如 TinyPNG）
2. 改用 WebP 格式
3. 上传前在本地缩小分辨率

### Q：如何批量修改卡片？
A：目前需要逐个编辑。考虑在未来版本中添加批量编辑功能。

### Q：删除卡片后能恢复吗？
A：不能。建议修改前先保存配置备份。

### Q：图标显示不对？
A：
1. 检查是否保存了配置
2. 清除浏览器缓存
3. 刷新页面（Ctrl+F5）

## 最佳实践

### 配置建议
1. **设置合理的颜色**：保证文字可读性
2. **图片要优化**：压缩后再上传
3. **定期备份**：保存配置文件副本
4. **定期更新**：及时添加新歌曲

### 内容建议
1. **卡片信息要精简**：控制在2-3行
2. **链接不超过5个**：太多影响美观
3. **图片清晰**：避免模糊或失真
4. **图标与链接匹配**：提高可识别性

### 安全建议
1. **定期修改密码**：不要使用默认密码
2. **谨慎分享链接**：配置和歌单页面可能被猜测
3. **定期备份**：防止意外数据丢失
4. **检查更新**：及时获取安全补丁
